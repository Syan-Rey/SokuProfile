# 東方非想天則 プロフィールメーカー

『東方非想天則』のプレイヤープロフィール画像をブラウザ上で作成・保存できるツールです。
本ツールは、コミュニティ内での交流を目的とした二次創作物です。

### 使い方
1. `index.html` をブラウザで開きます。
2. 各項目（名前、段位、デバイス、コメント等）を入力します。
3. キャラクターとカラー（1P〜8P）を選択すると、プレビューに反映されます。
4. 「画像として保存」ボタンで、作成したプロフィールをPNG形式で保存できます。
